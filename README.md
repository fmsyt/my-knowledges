# my-knowledges

## Requirements

- python <= 3.10

## Install on your server

```.bash
$ cd /path/to/my-knowledges
$ source venv/bin/activate
(venv) $ pip install pipenv
(venv) $ pipenv install
```

## Run server

```.bash
$ cd /path/to/my-knowledges
$ source venv/bin/activate
(venv) $ python manage.py runserver
```
